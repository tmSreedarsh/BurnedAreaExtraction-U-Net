# Burned Area Extraction using U-Net

This project uses a U-Net deep learning model to extract burned areas from satellite images using Landsat & Sentinel-2 data. The dataset is downloaded using the Google Earth Engine API.

## Tools Used
- Python
- PyTorch
- Google Earth Engine
- GDAL
- NumPy
- Matplotlib

## Folder Structure
- `data/`: Sample input data
- `models/`: U-Net model and checkpoints
- `notebook/`: Jupyter notebooks for training/visualization
- `scripts/`: Python scripts for training and prediction
- `results/`: Output maps and graphs
